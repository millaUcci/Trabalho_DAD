Algumas definições

primary key: é um valor unico que identifica cada dado em uma tabela

Tipos de campos

integer: campo numérico
varchar(tamanho): campo de texto com tamanho definido
boolean: campo com valor true ou false

Verificações:

notnull: campo não pode ser nullo (vazio)
unique: o valor do campo será unico
primary key: o campo receberá o valor da chave primária da tabela

Order by:

Order by (order por) serve para ordenar uma consulta de acordo com o valor de algum campo. Ex: eu crio uma consulta que trás todos os dados da tabela pessoa e eu ordeno o resultado pelo campo "nome" em ordem crescente (alfabética). Obs: o order permite ordenar os campos em ordem decrescente e ascendente (tanto campos varchar como os campos integer)
